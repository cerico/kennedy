# Clean Start
---
#### The plain and simple starter.

# Installation
---
Install and run: 
```
git clone https://github.com/cerico/kennedy
cd kennedy
npm install
npm run dev
```

# Write new post

```
npm run post
```

# Build 

```
npm run build
```
